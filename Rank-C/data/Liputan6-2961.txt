<title>toko swalayan di rawamangun terbakar, 10 mobil damkar diterjunkan news liputan6.com</title>
<content>patroli, jakarta sebuah toko swalayan yang berada di ruas jalan balai pustaka, rawamangun, jakarta timur, jumat malam terbakar. seperti ditayangkan patroli indosiar, sabtu 20/10/2018, api berkobar melahap sebuah toko swalayan di ruas jalan balai pustaka, rawamangun. api dengan cepat merambat ke lantai bawah toko swalayan ini. beruntung sebagian barang dagangan masih bisa diselamatkan.menurut petugas, kebakaran terjadi akibat hubungan pendek arus listrik di lantai dua. api baru berhasil dipadamkan petugas setengah jam kemudian setelah dikerahkan sebanyak 10 unit mobil pemadam kebakaran. tidak ada korban jiwa dalam peristiwa kebakaran ini. sementara itu, asap tebal mengepul dari dalam gudang tempat penyimpanan kapas di sebuah pabrik benang di jalur pantura plumbon, kabupaten cirebon, jumat malam. kobaran api menghanguskan hampir sebagian besar tumpukan kapas yang siap diproduksi menjadi benang. peristiwa kebakaran terjadi saat pabrik dalam keadaan produksi. di saat bersamaan, sejumlah pekerja tengah beraktivitas memperbaiki atap gudang dengan mesin las. beruntung seluruh karyawan dapat menyelamatkan diri dan tidak menimbulkan korban jiwa. diduga kebakaran yang menghanguskan hampir sepenuhnya tumpukan kapas ini berasal dari percikan las yang mengenai kapas. petugas pemadam kebakaran mengerahkan empat unit kendaraan dan satu unit eskavator. kerugian ditaksir mencapai ratusan juta rupiah. muhammad gustirha yunas</content>
